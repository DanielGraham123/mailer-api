Mailer API Documentation

Welcome to the Mailer API

In order to alleviate the stress of setting up a mail server each time we are builing a new website, why not use this mailer API that is already set up and ready to go.

It is built with Node.js, Express.js and Nodemailer using the tekshark mailing credentials to relay emails to the recipient.

Overview

Our Mailer API is built on Node.js and Express.js, leveraging SMTP for fast and reliable email delivery. Secure your requests with API keys and domain whitelisting while enjoying the flexibility to send both plain text and rich HTML emails.

API Endpoints

GET /readme

Renders the README in HTML format with dynamic values.

POST /send

Sends an email using the provided JSON payload.

Sample Payload:

{
  "fromHost": "Your Company",
  "replyTo": "user@example.com",
  "to": "recipient@example.com",
  "subject": "Your Subject",
  "text": "Plain text email content",
  "html": "<p>HTML email content</p>"
}

Setup Environment Variables

To integrate the Mailer API with a React frontend, ensure your .env file contains the following variables: 

REACT_APP_MAIL_SERVER=https://mailer.tekshark.pro
REACT_APP_MAIL_USER="The Site/Company Name"
REACT_APP_MAIL_KEY="Velte's mailer API key"
REACT_APP_EMAIL_TO=yousiteemail@yourdomain

Integrate with React

Integrate the Mailer API into any React project with a simple API call: 

const sendEmail = async () => {
  const response = await fetch(`${process.env.REACT_APP_MAIL_SERVER}/send`, {
    method: "POST",
    headers: {
      "Content-Type": "application/json",
      authorization: process.env.REACT_APP_MAIL_KEY,
    },
    body: JSON.stringify({
      fromHost: process.env.REACT_APP_MAIL_USER,

    }),
  });
  // Handle the response accordingly...
};
